# Ecommerce App Clientside

A React + Redux application for browsing products, adding them to cart, and checking out. It includes user registration and login.

## Features
- Browse products
- Add products to cart
- Checkout
- User registration and login

## Technologies
- React
- Redux Toolkit
- Redux query
- RESTful API
